Cones Section OEE Dashboard

This dashboard visualises Overall Equipment Effectiveness for the Cones section using a 3-factor model (no cavity term, unlike IML).

OEE Formula

Overall OEE = OEE_running × Shift Run % OEE_running = Availability × Performance × Quality = (run_hrs / shift_hrs) × (actual_speed / rated_speed) × (good / (good + rejected)) Overall OEE ≈ actual_good / (rated_target × total_slots / ran_slots)

Waterfall Steps

Shift Run %
ran_hrs / (slots × 12)
% of machine-shifts that actually ran
Availability
run_hrs / shift_hrs
Time lost to downtime within running shifts
Performance
actual_speed / rated_speed
Speed loss vs rated cones/hour for machine
Quality
good / (good + rejected)
Fraction of output that met spec

ETL Script

Run cones_oee_etl.py in the folder containing your daily Excel files:

py cones_oee_etl.py py cones_oee_etl.py --folder "C:\Cones OEE Files" py cones_oee_etl.py --output "C:\dashboard\cones_oee_data.json"

The script expects the date in the filename as DD.MM.YYYY or DD.MM.YY (e.g. 14.05.26.xlsx).

⚠ Each Excel file must have both a Shift tracker and OEE tracker sheet with the standard column layout.
